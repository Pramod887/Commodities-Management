# Commodities Management System

Frontend-only React + Vite app with role-based auth (mocked), protected routes, manager dashboard, product management, and light/dark mode using local state and mock data.

## Live Demo
- Deployed: https://commodities-management.netlify.app/

## Run Locally
1) Install deps: `npm install`
2) Start dev server: `npm run dev`
3) Open the printed local URL (default http://localhost:5173)

## Build
- Production build: `npm run build`
- Preview build: `npm run preview`

## Auth (Mocked)
- Manager: `manager@slooze.com` / `123456`
- Store Keeper: `store@slooze.com` / `123456`
